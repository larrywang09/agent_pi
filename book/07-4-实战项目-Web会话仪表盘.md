# 实战项目四：会话数据分析仪表盘（RPC + Web）

## 项目概述

前面两个项目都是 pi 内部的能力扩展。这个项目完全不同——我们将通过 **RPC 模式** 与 pi 通信，构建一个**外部 Web 仪表盘**，实时可视化和分析 pi 的会话数据。

通过这个项目，你将掌握：
- pi 的 RPC 协议（`RpcCommand` / `RpcResponse`）
- RPC 扩展 UI 协议（`RpcExtensionUIRequest` / `RpcExtensionUIResponse`）
- 通过 JSON-Lines over stdio 与 pi 进程通信
- 会话数据分析：Token 用量、成本、工具使用频率等
- WebSocket 实时推送会话事件

## pi RPC 客户端

```typescript
// src/pi-client.ts
import { spawn, type ChildProcess } from "node:child_process";
import { createInterface } from "node:readline";
import type {
    RpcCommand,
    RpcResponse,
    RpcExtensionUIRequest,
    RpcSessionState,
} from "@earendil-works/pi-coding-agent";

/**
 * pi RPC 客户端 — 通过 stdio JSON-Lines 与 pi 进程通信
 */
export class PiRpcClient {
    private process: ChildProcess;
    private rl: readline.Interface;
    private pendingRequests = new Map<string, {
        resolve: (value: RpcResponse) => void;
        reject: (error: Error) => void;
    }>();
    private eventHandlers = new Map<string, Set<(event: any) => void>>();
    private requestId = 0;
    private buffer = "";

    constructor(piPath: string = "pi", args: string[] = []) {
        this.process = spawn(piPath, ["--mode", "rpc", ...args], {
            stdio: ["pipe", "pipe", "pipe"],
            env: { ...process.env },
        });

        // 解析 stdout 上的 JSON Lines
        this.rl = createInterface({ input: this.process.stdout! });
        this.rl.on("line", (line) => {
            try {
                const msg = JSON.parse(line);

                // 响应
                if (msg.type === "response" && msg.command) {
                    const pending = this.pendingRequests.get(msg.id);
                    if (pending) {
                        this.pendingRequests.delete(msg.id);
                        pending.resolve(msg as RpcResponse);
                    }
                    return;
                }

                // 扩展 UI 请求
                if (msg.type === "extension_ui_request") {
                    this.emit("extension_ui_request", msg as RpcExtensionUIRequest);
                    return;
                }

                // 会话事件
                if (msg.type === "event") {
                    this.emit(msg.event.event, msg.event);
                    return;
                }
            } catch {}
        });

        this.process.on("exit", (code) => {
            console.error(`pi process exited with code ${code}`);
            this.emit("exit", { code });
        });
    }

    // ====================================================================
    // 发送 RPC 命令
    // ====================================================================

    async sendCommand<T extends RpcCommand["type"]>(
        command: T,
        params: Partial<RpcCommand & { type: T }> = {},
    ): Promise<RpcResponse> {
        const id = String(++this.requestId);
        const msg = { id, type: command, ...params };

        return new Promise((resolve, reject) => {
            const timeout = setTimeout(() => {
                this.pendingRequests.delete(id);
                reject(new Error(`RPC command '${command}' timed out`));
            }, 30000);

            this.pendingRequests.set(id, {
                resolve: (value) => { clearTimeout(timeout); resolve(value); },
                reject: (error) => { clearTimeout(timeout); reject(error); },
            });

            this.process.stdin!.write(JSON.stringify(msg) + "\n");
        });
    }

    // ====================================================================
    // 便捷方法
    // ====================================================================

    /** 发送提示 */
    async prompt(message: string): Promise<RpcResponse> {
        return this.sendCommand("prompt", { message });
    }

    /** 获取会话状态 */
    async getState(): Promise<RpcSessionState> {
        const response = await this.sendCommand("get_state");
        return response.data as RpcSessionState;
    }

    /** 获取可用模型列表 */
    async getAvailableModels(): Promise<any[]> {
        const response = await this.sendCommand("get_available_models");
        return (response.data as any)?.models ?? [];
    }

    /** 设置模型 */
    async setModel(provider: string, modelId: string): Promise<RpcResponse> {
        return this.sendCommand("set_model", { provider, modelId });
    }

    /** 获取会话历史 */
    async getSessionHistory(): Promise<any[]> {
        const response = await this.sendCommand("get_state");
        const state = response.data as any;
        return state?.messages ?? [];
    }

    /** 中止当前操作 */
    async abort(): Promise<void> {
        await this.sendCommand("abort");
    }

    // ====================================================================
    // 事件订阅
    // ====================================================================

    on(event: string, handler: (data: any) => void): () => void {
        if (!this.eventHandlers.has(event)) {
            this.eventHandlers.set(event, new Set());
        }
        this.eventHandlers.get(event)!.add(handler);

        return () => {
            this.eventHandlers.get(event)?.delete(handler);
        };
    }

    private emit(event: string, data: any): void {
        this.eventHandlers.get(event)?.forEach(h => h(data));
    }

    /** 关闭连接 */
    close(): void {
        this.process.stdin?.end();
        this.process.kill();
    }
}
```

## 数据分析引擎

```typescript
// src/dashboard.ts
import type { RpcSessionState } from "@earendil-works/pi-coding-agent";
import type { PiRpcClient } from "./pi-client.ts";

export interface SessionStats {
    sessionId: string;
    totalMessages: number;
    userMessages: number;
    assistantMessages: number;
    toolCalls: number;
    toolResults: number;

    // Token 统计
    totalInputTokens: number;
    totalOutputTokens: number;
    totalCacheReadTokens: number;
    totalCost: number;

    // 模型使用分布
    modelUsage: Record<string, number>;

    // 工具使用频率
    toolFrequency: Record<string, number>;

    // 时间统计
    startTime: string;
    duration: number; // 毫秒
}

export class SessionAnalyzer {
    private client: PiRpcClient;

    constructor(client: PiRpcClient) {
        this.client = client;
    }

    async collectStats(): Promise<SessionStats> {
        const state = await this.client.getState();
        const messages = state?.messages ?? [];

        const stats: SessionStats = {
            sessionId: state?.sessionId ?? "unknown",
            totalMessages: messages.length,
            userMessages: 0,
            assistantMessages: 0,
            toolCalls: 0,
            toolResults: 0,
            totalInputTokens: 0,
            totalOutputTokens: 0,
            totalCacheReadTokens: 0,
            totalCost: 0,
            modelUsage: {},
            toolFrequency: {},
            startTime: messages[0]?.timestamp
                ? new Date(messages[0].timestamp).toISOString()
                : "N/A",
            duration: messages.length > 1
                ? messages[messages.length - 1].timestamp - messages[0].timestamp
                : 0,
        };

        for (const msg of messages) {
            switch (msg.role) {
                case "user":
                    stats.userMessages++;
                    break;

                case "assistant": {
                    stats.assistantMessages++;

                    // Token 统计
                    if (msg.usage) {
                        stats.totalInputTokens += msg.usage.input || 0;
                        stats.totalOutputTokens += msg.usage.output || 0;
                        stats.totalCacheReadTokens += msg.usage.cacheRead || 0;
                        if (msg.usage.cost?.total) {
                            stats.totalCost += msg.usage.cost.total;
                        }
                    }

                    // 模型使用统计
                    const modelKey = `${msg.provider}/${msg.model}`;
                    stats.modelUsage[modelKey] = (stats.modelUsage[modelKey] || 0) + 1;

                    // 工具调用统计
                    const toolCalls = msg.content?.filter((c: any) => c.type === "toolCall") ?? [];
                    for (const tc of toolCalls) {
                        stats.toolCalls++;
                        stats.toolFrequency[tc.name] = (stats.toolFrequency[tc.name] || 0) + 1;
                    }
                    break;
                }

                case "toolResult":
                    stats.toolResults++;
                    break;
            }
        }

        return stats;
    }

    /** 实时追踪 — 订阅事件并更新统计 */
    async liveTracking(
        onStatsUpdate: (stats: SessionStats) => void,
        intervalMs: number = 2000,
    ): Promise<() => void> {
        // 初始数据
        const initialStats = await this.collectStats();
        onStatsUpdate(initialStats);

        // 定期刷新
        const timer = setInterval(async () => {
            try {
                const stats = await this.collectStats();
                onStatsUpdate(stats);
            } catch {}
        }, intervalMs);

        return () => clearInterval(timer);
    }
}
```

## Web 服务器

```typescript
// src/server.ts
import { createServer } from "node:http";
import { readFileSync, existsSync } from "node:fs";
import { join, extname } from "node:path";
import { WebSocketServer } from "ws";
import { PiRpcClient } from "./pi-client.ts";
import { SessionAnalyzer } from "./dashboard.ts";

const PORT = parseInt(process.env.PI_DASHBOARD_PORT || "3030");
const STATIC_DIR = join(import.meta.dirname, "static");

const MIME_TYPES: Record<string, string> = {
    ".html": "text/html; charset=utf-8",
    ".js": "application/javascript; charset=utf-8",
    ".css": "text/css; charset=utf-8",
    ".json": "application/json",
    ".svg": "image/svg+xml",
};

async function main() {
    // 启动 pi RPC 客户端
    const piPath = process.env.PI_PATH || "pi";
    const piClient = new PiRpcClient(piPath);
    const analyzer = new SessionAnalyzer(piClient);

    // 获取初始数据
    const initialStats = await analyzer.collectStats();

    // HTTP 服务器（提供静态文件）
    const server = createServer((req, res) => {
        const url = req.url === "/" ? "/index.html" : req.url!;
        const filePath = join(STATIC_DIR, url);

        if (!existsSync(filePath)) {
            res.writeHead(404);
            res.end("Not Found");
            return;
        }

        const ext = extname(filePath);
        const contentType = MIME_TYPES[ext] || "application/octet-stream";
        const content = readFileSync(filePath);

        res.writeHead(200, { "Content-Type": contentType });
        res.end(content);
    });

    // WebSocket 服务（推送实时数据）
    const wss = new WebSocketServer({ server });

    wss.on("connection", (ws) => {
        console.log("Dashboard client connected");

        // 发送初始数据
        ws.send(JSON.stringify({ type: "init", data: initialStats }));

        // 处理客户端请求
        ws.on("message", async (msg) => {
            try {
                const cmd = JSON.parse(msg.toString());

                switch (cmd.type) {
                    case "get_stats":
                        const stats = await analyzer.collectStats();
                        ws.send(JSON.stringify({ type: "stats", data: stats }));
                        break;

                    case "set_model":
                        await piClient.setModel(cmd.provider, cmd.modelId);
                        ws.send(JSON.stringify({ type: "success", message: "Model changed" }));
                        break;

                    case "prompt":
                        await piClient.prompt(cmd.message);
                        ws.send(JSON.stringify({ type: "success", message: "Prompt sent" }));
                        break;

                    case "get_models":
                        const models = await piClient.getAvailableModels();
                        ws.send(JSON.stringify({ type: "models", data: models }));
                        break;
                }
            } catch (error: any) {
                ws.send(JSON.stringify({ type: "error", message: error.message }));
            }
        });

        // 实时推送
        const stopTracking = await analyzer.liveTracking((stats) => {
            try {
                ws.send(JSON.stringify({ type: "stats", data: stats }));
            } catch {}
        });

        ws.on("close", () => {
            console.log("Dashboard client disconnected");
            stopTracking();
        });
    });

    server.listen(PORT, () => {
        console.log(`pi Dashboard running at http://localhost:${PORT}`);
        console.log(`Connected to pi via RPC (${piPath} --mode rpc)`);
    });
}

main().catch(console.error);
```

## 仪表盘功能

| 功能 | 实现方式 | 数据来源 |
|------|---------|---------|
| 消息统计 | WebSocket 实时推送 | RPC `get_state` |
| Token 用量 | 从 `AssistantMessage.usage` 统计 | RPC 事件流 |
| 成本追踪 | `usage.cost.total` 累计 | RPC 事件流 |
| 工具使用频次 | 统计 `toolCall.name` | RPC 事件流 |
| 模型分布 | 按 `provider/model` 分组 | RPC `get_state` |
| 实时会话事件 | 订阅 `message_end`/`turn_end` | RPC 事件流 |
